# HOUSE-PRICE-PREDITION
Predict house prices using machine learning regression.
Use a CSV dataset with area, rooms, location score, and price.
Python is used as the main programming language.
Pandas is used for data loading and preprocessing.
Seaborn is used to visualize feature correlations.
Matplotlib is used for basic plotting and graphs.
Linear Regression is used to model price prediction.
Features include area, number of rooms, and location score.
Model performance is evaluated using MSE and RÂ² score.
The project helps understand feature importance and pricing trends.
